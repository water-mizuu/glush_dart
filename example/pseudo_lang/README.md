# Pseudo-Programming Language Example

This example demonstrates how to build a custom programming language using the GLUSH parsing library. It features a hierarchical parse tree and a tree-walking interpreter.

## Language Features

The pseudo-language supports:
*   **Variable Declarations**: `let x = 10;`
*   **Assignments**: `x = x + 5;`
*   **Conditionals**: `if (cond) { ... } else { ... }`
*   **Loops**: `while (cond) { ... }`
*   **Functions**: `fn main() { ... }`
*   **Expressions**: Arithmetic (`+`, `-`, `*`, `/`) and Comparisons (`==`, `!=`, `<`, `>`, `<=`, `>=`).
*   **Built-ins**: `print(value)` for console output.

## Directory Structure

*   **`interpreter.dart`**: A tree-walking interpreter that executes the `ParseResult` tree generated by GLUSH.
*   **`main.dart`**: The example script that defines the grammar, parses a sample program, and runs it.

## How it Works

1.  **Parsing**: The `Grammar` is defined in `main.dart` using labels to identify key language constructs (e.g., `ifStmt:ifStatement`).
2.  **Structured Evaluation**: The `StructuredEvaluator` converts flat parsing marks into a nested `ParseResult` tree based on those labels.
3.  **Execution**: The `Interpreter` walks this tree. It uses an `Environment` to manage variable scopes and recursively visits nodes to evaluate expressions and execute statements.

## Running the Example

From the project root:

```bash
dart example/pseudo_lang/main.dart
```
